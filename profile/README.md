## Public versus private

- **Public** (CC BY-NC-ND 4.0): semantic standard, audit-record JSON Schema, Calibration Class prompts (C1/C2/C3), entity list, harness reference implementation.
- **Private** (reserved): synthesis coefficients (alpha, beta, omega), asymmetric tax weights, decay parameters, verdict thresholds, 5-year projection formula, and the `metaognition` skill that contains them. These are operational implementation details and are not necessary for academic verification.

## Why the split

The public layer establishes the *standard* — anyone can adopt the schema, run audits, and report results. The private layer is the licensed scoring instrument operated by the author and licensees. Reverse-engineering of private parameters is prohibited under the licence and the UK IPO trademark filings.
